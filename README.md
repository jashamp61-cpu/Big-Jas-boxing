# BigJasBoxing — Web App (React + Vite + Tailwind)


## What's included


- React + Vite project scaffold
- Tailwind CSS configuration
- `public/logo.png` — your Big Jas logo (included)
- Sponsorship content and donate links (Venmo, PayPal, Chime)

## Quick start

1. Install dependencies:

```bash
npm install
```

2. Development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm run preview
```

## Deploy

- Deploy to Vercel: drag-and-drop the repo or connect your GitHub repo and set framework to "Vite".
- Deploy to Netlify: connect the repo and set build command `npm run build` and publish directory `dist`.

## Notes

- The project uses simple donation links (Venmo/PayPal). For integrated payments use Stripe or PayPal SDK and follow their docs.

## Domain

Set up your domain `bigjasboxing.com` with your hosting provider (Vercel/Netlify) DNS settings.
